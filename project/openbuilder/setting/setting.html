봇 복사

시나리오, 스킬 그리고 엔티티 데이터들을 JSON 형태로 내보내고 가져올 수 있습니다.
초기화 후 가져오기는 데이터를 모두 삭제하고 넣는 것이며, 덮어쓰기는 기존 데이터를 지우지 않고 덮어쓰기만을 합니다.
덮어쓰는 경우 각 시나리오, 블럭, 엔티티의 고유값이 유지됨을 주의해주세요.

봇 삭제

자동삭제까지 남은 시간 : 약 5일
#if($params.get('del'))
봇의 마스터가 봇을 삭제합니다. 동의 또는 반대 하지 않는 경우, 7일 뒤에 봇이 자동으로 삭제됩니다.
#else
삭제 대기중입니다. 모든 정식 작업자가 찬성하는 경우 바로 삭제되고, 한명의 반대자라도 있다면 삭제가 취소됩니다. 별도로 찬성/반대하지 않는 경우 7일 뒤에 자동 삭제 처리됩니다.
#end
