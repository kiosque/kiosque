권한 체계 변경 알림
기존의 권한 체계가 아래와 같이 새롭게 변경되었습니다.
변경된 권한 체계
  1. 1. 새로운 마스터 권한이 생겼습니다. 봇을 만든 사용자가 마스터가 됩니다.
  2. 2. 마스터 권한은 기존 관리자가 2명 이상인 경우 수동 지정이 필요합니다.
  3. 3. 기존의 '오너' 등급은 자동으로 '정식 작업자' 권한으로 변경됩니다.
  4. 4. 기존의 '디자이너' 등급은 자동으로 커스텀 권한인 '디자이너' 권한으로 변경됩니다.
  5. 5. 마스터는 커스텀 권한을 생성할 수 있습니다. 커스텀 권한은 해당 봇에서만 유효합니다.
  6. 6. 커스텀 권한은 오픈 빌더 내 기능들을 부분적으로 조합하여 구성할 수 있습니다.
